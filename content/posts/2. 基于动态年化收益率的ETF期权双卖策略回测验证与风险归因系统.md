基于 159919 ETF 期权的名义现金备付双卖策略（100% 名义仓位 / 0 杠杆）

## 3.1 项目定位与背景

本项目围绕 ETF 期权动态年化收益率双卖策略展开。策略的基础结构是同时卖出虚值 Call 和虚值 Put，收取两侧权利金；但本项目并不是固定虚值、固定 Delta 或固定到期日的机械双卖，而是把“当前权利金能够提供的名义年化收益率”作为开仓温度计，用收益率门槛决定是否值得卖出期权。

在 100% 名义仓位、0 杠杆前提下，回测中出现了“约3%最大回撤换取约40%的总收益”的参数组合。这个策略不是只追求高收益，而是在收益、回撤、仓位占用和退出规则之间做了平衡。

## 3.2 策略结构：两腿动态双卖

策略由两条卖方腿组成：卖出虚值Call，收取上方权利金；卖出虚值Put，收取下方权利金。开仓时先根据目标虚值比例TargetOTMPct定位基础Call/Put行权价，再计算该组合的名义年化收益率。如果收益率低于MinNotionalYield，则放弃开仓。

核心公式如下：
```
认购行权价 = ETF现价 × (1 + TargetOTMPct)
目标认沽行权价 = ETF现价 × (1 - TargetOTMPct)
名义年化收益率 = (Call权利金 + Put权利金) / ETF现价 × 365 / DTE
单组名义本金 = ETF现价 × 10000
开仓组数 = floor(账户资金 × PositionRatio / 单组名义本金)
```

## 3.3 核心参数设计

| **参数**           | **含义**   | **当前搜索/测试范围**          | **策略作用**              |
| :--------------- | :------- | :--------------------- | :-------------------- |
| PositionRatio    | 名义仓位比例   | 1.0                    | 代表 100% 名义现金备付，0 杠杆   |
| TargetOTMPct     | 基础开仓虚值比例 | 1% - 8%，当前最佳 2%        | 决定初始 Call/Put 离现价的距离  |
| MinNotionalYield | 最低名义年化门槛 | 5% - 30%，当前最佳 25%      | 收益率低于门槛时不交易           |
| MaxLossMultiple  | 权利金止损倍数  | 1.0 - 3.0，当前最佳 1.0     | 控制权利金扩大后的极端损失         |
| TakeProfitPct    | 止盈比例     | 20% - 90%，当前最佳 60%     | 权利金回落到一定比例后获利退出       |
| ExitDTE          | 临期退出天数   | 2 - 7 天                | 避免临近到期 Gamma 风险和流动性风险 |
| VolExpandTrigger | 高波防守触发倍数 | 1.2 - 3.0，当前最佳 2.0/2.8 | 当基准收益显著高于历史均值时触发动态外扩  |
| MaxDynamicOTM    | 最大动态退守虚值 | 5% - 9%，当前最佳 5%        | 限制外扩幅度，防止收益被过度稀释      |

## 3.4 数据工程与期权链构建

策略回测的第一步是构建可用的分钟级期权链。项目从 ETF 分钟线、期权静态信息和各期权合约分钟行情出发，将标的价格与期权价格按时间戳对齐，并为每个有效时间点构建多到期日的 Call/Put 期权链。

每个期权合约记录 ID、合约代码、Call/Put 类型、行权价和到期时间；每个分钟报价记录价格、DTE、行权价，并通过 Black-Scholes 计算 IV、Delta、Gamma。构建完成后，系统按到期日分组，并将 Call 与 Put 分别按行权价排序，为后续目标虚值定位、持仓追踪和 T 型报价日志提供基础。

```
标的：159919 ETF 期权；回测起点：2023-01-03 09:31；初始资金：1,000,000元。
DTE 过滤：只选择15到60天之间的期权链，避免过近到期和过远期限带来的结构偏差。
价格过滤：过滤价格低于0.0010的合约，减少无效报价对回测的干扰。
并发加载：通过多线程加载期权行情，并在构建Tick时预计算Greeks。
```


## 3.5 名义年化收益率过滤

策略核心是先判断卖方组合提供的权利金是否足以补偿风险。系统先在目标虚值附近寻找Call和Put，并计算该组合的名义年化收益率：

```
(Call权利金 + Put权利金) / ETF现价 × 365 / DTE
```

如果该收益率低于MinNotionalYield，则跳过当前期权链，不开仓。
这一设计使策略在低波动、低权利金、风险收益不匹配的环境中自动空仓。
在当前最佳组合中，最低名义年化门槛为 25%。这意味着策略不是在每个交易日都强行卖出期权，而是在期权价格足够贵、名义年化收益率达到要求时才开仓。

## 3.6 高波防守外扩：Chain Walking

为了避免在高波动环境中仍然贴近现价开仓，策略维护了一个10天的基准年化收益率历史均线。当当天基准年化收益率大于历史均线的VolExpandTrigger倍时，系统认为市场进入高波或异常权利金环境，触发高波防守外扩。
触发外扩是在期权链上同步向外移动：认购腿向更高行权价移动，认沽腿向更低行权价移动。每移动一步，都必须同时满足三类约束：有有效报价、未超过最大退守虚值MaxDynamicOTM、移动后名义年化收益率仍不低于MinNotionalYield。
所以，当市场给出异常高的权利金时，我们不急于在原始2%虚值附近承担风险，而是把一部分权利金让渡给安全边际，向更远虚值退守；如果交易所没有更远可交易合约，或者外扩后收益不达标，则宁可空仓，不为了交易而交易。
正常状态：按基础TargetOTMPct找到虚值Call/Put，并验证最低年化收益。
高波状态：若当前基准收益率 >= 10日均线 × VolExpandTrigger，则逐步外扩。
策略边界：外扩不能超过MaxDynamicOTM；收益率不能低于MinNotionalYield。
熔断保护：若触发防守后某一侧仍被迫贴近现价，则跳过该期权链，避免极端单边风险。

## 3.7 开仓、平仓与风控规则

目前的开仓规则中，策略只在10:00到14:00之间寻找交易机会，同一天平仓后不再重复开仓，并且如果未来5天内存在分红除权事件则直接跳过开仓。仓位按照100% 名义现金备付计算，即每组名义本金约为 ETF现价 × 10000，系统根据当前账户资金计算可开仓组数。

| **退出原因** | **触发条件**                                              | **策略意义**          | **面试表达**                |
| :------- | :---------------------------------------------------- | :---------------- | :---------------------- |
| 目标止盈     | currentCost <= netEntryCredit × (1 - TakeProfitPct)   | 权利金衰减后兑现收益        | 不是等到到期，而是主动收割时间价值/波动率回落 |
| 提前防守     | currentCost > netEntryCredit × 1.1                    | 利润垫被吃掉后撤退         | 用“利润垫耗尽”替代被动扛单          |
| 权利金止损    | currentCost >= netEntryCredit × (1 + MaxLossMultiple) | 极端情况下的最后止损阈值      | 避免亏损无上限扩大               |
| 到期规避     | realDTE <= ExitDTE                                    | 规避临近到期 Gamma 风险   | 不把临期尾部风险留到账户里           |
| 分红规避     | 进入分红相关窗口                                              | 规避除权导致的期权链和标的价格扰动 | 考虑国内 ETF 期权真实交易约束       |

## 3.10 当前不足与后续优化方向

| **优化方向**      | **当前状态**            | **后续价值**                     |
| :------------ | :------------------ | :--------------------------- |
| 保证金模型         | 当前按 100% 名义现金备付计算仓位 | 可进一步比较现金备付与保证金占用下的资金效率       |
| Vega/Theta 暴露 | 当前记录 IV、Delta、Gamma | 补充 Theta/Vega 后可解释收益来源与波动率风险 |
| 样本外验证         | 当前特定组合效果较好          | 拆分训练/验证区间，降低参数过拟合质疑          |
| 极端行情压力测试      | 已有止损和防守撤退逻辑         | 验证快速上涨/下跌、IV 跳升、分红除权下的风险边界   |
